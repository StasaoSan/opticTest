# Quiz Cards — тренажёр для подготовки к аккредитации

Статическое веб-приложение для прохождения тестов с вопросами и вариантами ответов. Создавалось для подготовки к аккредитации по медицинской оптике.

## Возможности

- **Три режима прохождения** — по порядку из общего пула, только избранные, только вопросы с ошибками
- **Сохранение сессии** — выйдя в меню и вернувшись, продолжаешь с того же вопроса и с тем же временем
- **Избранное** — помечаешь вопросы звёздочкой прямо во время теста, потом проходишь их отдельно
- **Ошибки** — неверные ответы накапливаются автоматически; правильный ответ в режиме «Мои ошибки» убирает вопрос из списка
- **Таймер** — обратный отсчёт (настраивается в `questions.json`), с подсветкой при малом остатке времени
- **Счётчик** — показывает `верно / отвечено` в реальном времени
- **Перемешивание ответов** — порядок вариантов случаен при каждом вопросе

## Структура файлов

```
index.html       — разметка (не трогай)
style.css        — стили (не трогай)
app.js           — логика (не трогай)
questions.json   — вопросы и настройки (редактируй здесь)
```

## Формат questions.json

```json
{
  "title": "Название теста",
  "timeLimit": 3600,
  "questions": [
    {
      "question": "Текст вопроса?",
      "answers": [
        { "text": "Вариант А", "correct": false },
        { "text": "Вариант Б", "correct": true  },
        { "text": "Вариант В", "correct": false },
        { "text": "Вариант Г", "correct": false }
      ]
    }
  ]
}
```

| Поле | Тип | Описание |
|---|---|---|
| `title` | string | Название теста, отображается в меню |
| `timeLimit` | number \| null | Лимит времени в секундах на весь тест. `null` — без ограничения, таймер считает вверх |
| `questions` | array | Массив вопросов |
| `question` | string | Текст вопроса |
| `answers` | array | Варианты ответов, минимум 2, максимум 6 |
| `answers[].text` | string | Текст варианта |
| `answers[].correct` | boolean | `true` у одного правильного варианта |

## Режимы теста

### Следующие вопросы
Вопросы из общего пула идут **по порядку**, по 80 за раз. Позиция сохраняется — следующий запуск начнётся с вопроса 81, затем с 161 и т.д. Когда пул заканчивается, переход к началу.

Кнопка **«Сбросить прогресс пула»** возвращает счётчик к вопросу 1.

### Избранное
Проходишь только вопросы, отмеченные звёздочкой (★). Кнопка доступна после добавления хотя бы одного вопроса в избранное.

### Мои ошибки
Проходишь вопросы, на которые ранее был дан неверный ответ. При правильном ответе вопрос автоматически убирается из списка ошибок.

## Деплой

Приложение полностью статическое — серверная логика не нужна. Достаточно разместить четыре файла на любом хостинге:

**GitHub Pages:**
```
git add .
git commit -m "init"
git push
# включи Pages в настройках репозитория (Settings → Pages → branch: main)
```

**Netlify / Vercel:** перетащи папку проекта в браузере — сайт готов.

> **Важно:** приложение загружает `questions.json` через `fetch()`, поэтому открытие `index.html` напрямую как файл (`file://`) не работает. Для локальной разработки запусти любой локальный сервер:
> ```
> npx serve .
> ```
> или используй расширение **Live Server** в VS Code.

## Локальные данные

Прогресс хранится в `localStorage` браузера:

| Ключ | Что хранит |
|---|---|
| `quiz_fav` | Индексы вопросов в избранном |
| `quiz_wrong` | Индексы вопросов с ошибками |
| `quiz_offset` | Текущая позиция в пуле |
| `quiz_session` | Активная прерванная сессия |

Данные привязаны к домену. При смене хостинга или очистке браузера прогресс сбрасывается.
